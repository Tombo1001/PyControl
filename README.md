# PyControl
A tools written in Python3 design to complete actions on the push of a GPIO connected button.

- The button: https://www.amazon.co.uk/Monk-Makes-Button-Latching-Individual/dp/B08126PF5R/ref=sr_1_2?dchild=1&keywords=squid+button&qid=1607815550&sr=8-2

## Current button actions:

- [IFTTT](https://ifttt.com/) webhooks
- [Virsh](https://linux.die.net/man/1/virsh) device attachment - used as a very rudimentary KVM attachment method for libvert VMs.
